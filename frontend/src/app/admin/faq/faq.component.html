Here is collected few frequently asked questions and answers for them
  • How to create object?
    Just try this link
  • How to create command for my object?
    You need to click on edit-icon against name of your object here
  • How to update base URL for all my commands?
    You can do this here
    Also you can update timeouts, headers and cookies of your commands
  • How to run my tests?
    Here click on button "Tests file", chose file and click on button "Run tests"
  • How simple test looks like?
    Something like this: 
              
            BEGIN
              createUserResponse = CREATE USER {{"{"}}"name": "Joe"}

              ASSERT createUserResponse.status EQUALS ok

              deleteUserResponse = DELETE USER ${{"{"}}createUserResponse.hash}

              ASSERT deleteUserResponse.status EQUALS ok

              userResponse = GET USER ${{"{"}}createUserResponse.hash}

              ASSERT userResponse.status EQUALS error
              ASSERT userResponse.data EQUALS user-not-found
            END

            BEGIN
              createUserResponse = CREATE USER {{"{"}}"name": "Nick"}

              ASSERT createUserResponse.status EQUALS ok

              userStatResponse = STAT USER {{"{"}}"hash": "${{"{"}}createUserResponse.hash}"}

              ASSERT userStatResponse.status EQUALS ok
              ASSERT_TRUE userStatResponse.data.exists
              ASSERT_FALSE userStatResponse.data.deleted
            END
  • Why my account was deleted on next day after registration?
    This is basic rule of this project - each account has no limits of usage but after 24 hours it will be deleted.
    If community will show interest to this project I will add verification of account; verified accounts won't be deleted
